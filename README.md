# covid-19-alert

| 정부에서 제공하는 코로나 지역별 확진자 통계 Open API를 이용해서 슬랙채널 알람을 주는 프로그램

## 설치
```
npm install
```

## 실행
* 테스트 환경 
```
node index.js --env=test
```
* 서비스 환경 
```
node index.js --env=prod
```
